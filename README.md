# NeuralSpark
类脑脉冲神经网络原型 | 探索智能的涌现机制

## 关于这个项目
现在的大模型，本质是靠海量数据做统计拟合，和生物大脑的工作逻辑完全是两码事。那如果我们真的按大脑的底层机制，搭一套由简单神经元组成的动态网络，能不能不靠监督训练，就让智能、记忆、甚至更复杂的认知功能，自己从里面长出来？

## 核心
1.  **严格遵循生物机制的神经元**：用计算神经科学通用的LIF（泄漏积分点火）模型，还原了生物神经元的膜电位泄漏、阈值点火、绝对不应期这些核心电生理特性，不是随便写个“开关”糊弄事。
2.  **真实的突触学习规则**：实现了STDP（脉冲时序依赖可塑性）——这就是大脑里形成记忆和学习的核心机制，完全靠神经元脉冲的发放时序，无监督地自动更新突触权重，不靠反向传播。
3.  **兴奋-抑制（E-I）平衡**：严格区分兴奋性和抑制性神经元，这是大脑皮层能稳定工作、形成功能环路的核心，也是现在很多SNN项目容易忽略的关键点。
4.  **事件驱动的稀疏计算**：只有神经元发放脉冲的时候才触发计算，天然符合大脑的低功耗、稀疏编码特性，和ANN全量更新的模式完全不一样。
5.  **全链路可观测**：能追踪每一个神经元的膜电位变化、每一次脉冲的传递路径，方便观察和量化涌现出来的动态行为，而不是一个黑盒。

## 和主流方案的区别
- 跟Transformer/传统ANN比：完全不是一个逻辑。我们不做序列预测、不做统计拟合，核心是模拟神经环路的动态交互，探索的是「智能是怎么来的」，而不是「怎么把任务做对」。
- 跟Brian2、NEST这些专业神经仿真框架比：那些框架更偏向于做生物环路的精细仿真，很多还是套着ANN的监督训练范式。我这个更聚焦于「涌现」，从最基础的单元搭起，做轻量化的探索实验，门槛更低，可观测性更强。

## 现在有什么
- 核心LIF神经元模型的完整实现
- 基于STDP的可塑突触
- 最小的兴奋-抑制闭环环路演示
- 单步时序仿真的完整调度逻辑

## 接下来的计划
- 先把10万级神经元的网络跑通，验证E-I平衡和稀疏编码的稳定性
- 做几个基础的无监督学习实验，验证记忆和适应性行为的形成
- 优化大规模仿真的效率，同时保留完整的可观测性
- 做一套可视化的工具，能直观看到网络里的脉冲流动和环路形成
